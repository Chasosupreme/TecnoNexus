¿Quiénes Somos?

El proyecto tiene como objetivo la implementación de una red cliente-servidor que integre dos servicios esenciales para la gestión y administración de la infraestructura tecnológica de una organización: File Server y DNS. La configuración de estos servicios permitirá optimizar la gestión de archivos dentro de la red y garantizar un adecuado funcionamiento de la resolución de nombres, lo cual es crucial para la comunicación y acceso a recursos en una red empresarial.

El proceso de implementación se centra en la instalación y configuración del servidor, así como en la creación de un entorno de dominio que facilite la administración centralizada de los recursos y usuarios a través de Active Directory. Esto permitirá una mayor seguridad, control de acceso y organización de los recursos de la red. Además, se pretende configurar el servicio de DNS para asegurar que todos los dispositivos en la red puedan comunicarse de manera eficiente, resolviendo correctamente los nombres de dominio.

La ejecución de este proyecto no solo proporcionará una infraestructura de red más organizada y segura, sino que también permitirá al equipo de trabajo adquirir una experiencia valiosa en la administración de servidores y servicios de red. A través de la implementación de estos servicios, se busca mejorar la conectividad, el acceso a archivos compartidos y la resolución de problemas relacionados con la red, lo cual es fundamental en cualquier entorno de TI moderno.

Además, se incluirán herramientas de validación y disponibilidad como tracert, nslookup, ping, entre otras, para asegurar el buen funcionamiento de la red y de los servicios configurados. De esta manera, el proyecto contribuirá a la mejora del rendimiento y la seguridad de la red en la organización, preparando a los participantes para futuros desafíos en el campo de la tecnología.

1.3 Formulación del Problema

En muchos entornos institucionales y empresariales, la infraestructura tecnológica carece de una administración centralizada, lo que genera desorganización en la red: equipos conectados sin control, ausencia de políticas de seguridad, usuarios sin credenciales personalizadas, y recursos compartidos sin medidas de protección. Esta falta de estructura compromete tanto la eficiencia operativa como la seguridad de la información.

Además, la inexistencia de servicios esenciales como un servidor DNS adecuadamente configurado dificulta la resolución de nombres dentro de la red, generando demoras y errores frecuentes en la comunicación entre dispositivos. De igual forma, la falta de un sistema de archivos centralizado impide una correcta gestión y respaldo de los documentos compartidos.

Frente a esta problemática, surge la necesidad de implementar una solución técnica basada en servidores, que incluya servicios como DNS, Active Directory y File Server. Esta propuesta permitirá organizar los recursos de red, establecer un modelo de dominio con usuarios y permisos definidos, y facilitar un entorno seguro, estable y escalable para el manejo de la información y la conectividad entre equipos.

¿Es posible transformar una red desorganizada y vulnerable en una infraestructura robusta, segura y eficiente mediante la correcta implementación de servicios de servidor?

1.5 Alcance del Proyecto

Este proyecto tiene como alcance el diseño, instalación y configuración de una red cliente-servidor dentro del Colegio Montferri, orientada a mejorar la organización, seguridad y administración de los recursos informáticos institucionales. La solución incluye la implementación de servicios como Active Directory, DNS y File Server, los cuales serán instalados y gestionados desde un servidor central con sistema operativo Windows Server.

La red contará con un dominio estructurado que permitirá el control de usuarios, permisos, políticas de grupo (GPO) y el acceso compartido a recursos mediante un servidor de archivos. Además, se configurará un servicio DNS interno para la resolución de nombres y una estructura jerárquica de Unidades Organizativas (OUs).

El proyecto está dirigido a un entorno educativo, específicamente a la sala de sistemas del colegio. Se implementará una red cliente-servidor con servicios básicos de administración y control de usuarios. El sistema será funcional y operativo para mejorar la administración de los recursos.

Entre los aspectos que se cubrirán se encuentran:

  • Configuración física y lógica de la red cliente-servidor en ambiente controlado.
  • Instalación de Windows Server y activación de roles esenciales (DNS, File Services).
  • Creación de usuarios, grupos y unidades organizativas.
  • Aplicación de políticas de grupo básicas para el control de accesos y recursos.
  • Documentación técnica del proceso de diseño, instalación y evaluación del sistema implementado.

Este proyecto busca implementar una solución funcional y práctica, enfocada en mejorar la organización y seguridad de la red en la sala de sistemas del colegio.
